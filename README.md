CosmeBun

A website that takes in data like chemicals commonly found in makeup, and shows the user how that reacts to skin through reactions/how it would permeate the epedermal layer of the skin.

It was trained on qsar and assay data, so that it could be as accurate as possible. It's generally very hard to find data regarding chemical reactions to skin.

Lovable was used just to make the website pretty because I was more focused on the ML aspect than anything else.

Try it: https://cosmebun.lovable.app
